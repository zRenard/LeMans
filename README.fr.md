# LeMans
Garmin Watch Face inspiré par les couleurs de la Ford GT40 LeMans

Available on Garmin ConnectIQ : https://apps.garmin.com/fr-FR/apps/000fe6f8-bb3b-4d39-b3c4-cee8745b48d9

# Possibilité de configuration

## Affichage
* Notifications (On/Off)
* Batterie (On/Off)
* Affiche la batterie uniquement en dessous d'un niveau critique

## Affichage du niveau de la batterie
* Valeur minimum pour l’affichage de l’alerte
